Software Form 1042-S - Income Types - Income Code :

To report the Tax Treaty based Income in Form 1042-S, you have to select the Income Type and the Income Code.

Some common Form 1042-S Income Types and their Income Codes;

Scholarships and Fellowship Grants - Income Code (15)

Capital Gains Distributions - Income Code (36)

Gambling Winnings - Income Code (28)

Earnings as an Artist or an Athlete - Income Code (20)

Compensation during Studying and Training - Income Code (19)

Compensation for Teaching - Income Code (18)

Compensation for Dependent Personal Services - Income Code (17)

Compensation for Independent Personal Services - Income Code (16)
