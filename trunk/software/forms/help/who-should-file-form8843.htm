Should I file Form 8843?

Form 8843 should only be filed by Students (for upto five years) and Teachers, Trainees (for upto two years), to claim their Tax Exemption Status.

H1B, L1, TN visa holders and other visa holders who are Non Resident Aliens should only file Form 1040NR or Form 1040NR-EZ. They should not file Form 8843.
